# Tugas Pertemuan 2: Flow Control Kotlin

Repositori ini berisi kumpulan program dasar Kotlin untuk memenuhi tugas mata kuliah **Pemrograman Mobile**.

## Isi File
1. `BilanganGenap.kt` : Menampilkan angka genap dari sebuah *array*.
2. `KelipatanAngka.kt` : Menampilkan angka kelipatan 3 dan 5 dari rentang 1-100.
3. `KlasifikasiAngka.kt` : Mengevaluasi angka (lebih/kurang dari 50) dan kelipatan 5.
4. `PerbandinganString.kt` : Membandingkan kesamaan dua buah teks secara *case-sensitive*.
5. `KlasifikasiUmur.kt` : Mengkategorikan umur berdasarkan rentang angka (`in`).
